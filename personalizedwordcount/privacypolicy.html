Privacy policy

Personalized Word Count's use and transfer to any other app of information received from Google APIs will adhere to Google API Services User Data Policy, including the Limited Use requirements.

Personalized Word Count will run its custom sidebar upon initialization and show pop-ups when the user prompts it to do so by clicking certain buttons.

Read the following for more information on the processing of your data:

Access

Personalized Word Count only has access to the document you open in it. The add-on is able to access all content in the file.

Use

  1. Word counting
  2. Text analysis to find citations, tables or other types of content specified so that they can later be highlighted through the auto-highlight feature
  3. Scanning the highlighting in your text to remove all yellow highlighting when prompted to do so

Storage

No data from your file is stored except for your preferences in the types of text that the auto-highlight feature should target. Still this data is not accessible by the creator and it is document specific, meaning you can have different feature settings on each document you use Personalized Word Count on.
